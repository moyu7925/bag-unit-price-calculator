# 单价计算器

一个简单易用的袋子单价计算工具，支持多种材质计算和模板管理。

## 功能特点

- 计算袋子单价
- 支持多种材质（铜板、橡胶等）
- 模板管理功能
- 数据持久化保存
- 简洁直观的界面

## 快速开始

### 运行程序

```powershell
# 方法1：使用启动脚本
python 启动计算器.py

# 方法2：直接运行
python 单价计算器.py

# 方法3：运行已打包的EXE
单价计算器.exe
```

### 打包成EXE

```powershell
# 使用打包脚本
.\打包.bat

# 或手动打包
pyinstaller 单价计算器.spec
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `单价计算器.py` | 主程序文件 |
| `单价计算器.spec` | PyInstaller打包配置 |

| `不能删除的数据文件.json` | 用户数据文件 |

| `启动计算器.py` | 快速启动脚本 |

| `打开模拟界面.bat` | 打开模拟界面 |
| `项目总览.md` | 项目概述文档 |

## 使用方法

1. 运行程序：`python 单价计算器.py`
2. 输入袋子参数（长度、宽度、厚度等）
3. 选择材质类型
4. 点击计算按钮
5. 查看计算结果
6. 可保存为模板供下次使用

## 数据文件

程序使用 `不能删除的数据文件.json` 存储用户数据和模板，请勿删除此文件。

## 打包说明

使用PyInstaller打包成独立EXE文件：

```powershell
# 安装PyInstaller
pip install pyinstaller

# 打包
pyinstaller 单价计算器.spec

# 或使用批处理脚本
.\打包.bat
```

打包后的EXE文件位于 `dist/` 目录。
